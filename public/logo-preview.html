Note: To see the PNG versions, you need to run one of the conversion scripts:

  • Direct SVG creation: npm run convert-logo:direct (creates SVG files that can be manually converted to PNG)
  • Canvas-based conversion: npm run convert-logo:canvas (requires Node.js with canvas package)
  • PowerShell script: .\scripts\convert-logo.ps1 (requires Inkscape)
  • Node.js script with sharp: npm run convert-logo (requires Node.js with sharp and fs-extra packages)

If you're having issues with PNG conversion: Use the direct SVG method and manually convert the SVGs to PNGs using a browser or image editor.
